ADMIN GUIDE
매거진 자동번역 운영 가이드
매거진 게시글을 복제한 뒤 각 언어 탭에서 자동번역을 실행하고, 검수 후 저장·게시하는 전체 플로우를 안내합니다.
복제 › 언어 탭 선택 › 자동번역 실행 › 번역 결과 확인 › 저장 › 게시 상태 변경
01
매거진 복제
매거진 목록 → 복제 버튼
SCREENSHOT 01
매거진 목록에서 번역할 원본 게시글을 찾아 복제합니다.
  • ✓BO 매거진 목록 화면 접속
  • ✓복제할 게시글 행의 [복제] 버튼 클릭
  • ✓목록 상단에 복사본이 생성됨 확인
💡 복제된 글은 원본 내용을 그대로 복사합니다. 언어별로 따로 복제할 필요 없이, 한 번만 복제한 뒤 언어 탭을 전환하며 번역합니다.
02
복제 글 진입
복제된 글 클릭 → 에디터 열기
SCREENSHOT 02
복제된 글을 클릭하여 에디터 화면으로 진입합니다.
  • ✓목록에서 복제된 글의 제목 클릭
  • ✓에디터 상단 언어 탭 위치 확인
  • ✓기본 탭 영어(EN) 선택 상태로 열림
💡 복제 직후에는 모든 언어 탭의 본문이 원본과 동일합니다. 자동번역 전에 내용을 수정하지 마세요.
03
언어 탭 선택
에디터 상단 → 번역 대상 언어 탭 클릭
SCREENSHOT 03
에디터 상단 언어 탭에서 번역할 언어를 클릭합니다.
  • ✓지원 언어: EN · JA · ZH · ZH-TW · ID
  • ✓번역 대상 언어 탭 클릭 → 탭 활성화 확인
  • ✓탭 전환 후 에디터 본문이 해당 탭으로 전환됨
💡 EN(영어)은 원본 언어로 자동번역 없이 직접 작성하거나 원본을 그대로 사용합니다. JA / ZH / ZH-TW / ID 탭에서 자동번역을 실행합니다.
04
자동번역 실행
에디터 툴바 → [자동번역] 버튼
SCREENSHOT 04
에디터 툴바의 [자동번역] 버튼을 클릭하여 번역을 시작합니다.
  • ✓에디터 툴바 → [자동번역] 클릭
  • ✓확인 팝업 표시됨 → 내용 확인
  • ✓팝업의 [자동번역] 버튼 클릭
⚠️ 번역 중에는 페이지를 이탈하거나 다른 탭을 클릭하지 마세요. 번역이 중단될 수 있습니다.
05
번역 결과 확인
노출 제목 · 에디터 본문 · 브랜드명 검수
SCREENSHOT 05
번역 완료 후 에디터 본문과 노출 제목을 검수합니다.
  • ✓노출 제목: 번역 언어로 자동 전환 — 어색하면 직접 수정 가능
  • ✓에디터 본문: 우측 미리보기 패널에서 확인
  • ✓브랜드명 · 수치 · 화폐 단위 올바르게 번역됐는지 확인
💡 상품/스토어 위젯 영역은 번역 대상에 포함되지 않습니다. 에디터 본문만 번역됩니다.
06
저장
우측 상단 → [저장] 버튼 클릭
SCREENSHOT 06
번역 검수 후 우측 상단 [저장] 버튼을 클릭합니다.
  • ✓화면 우측 상단 [저장] 버튼 클릭
  • ✓목록으로 돌아가려면 [목록으로] 클릭
💡 다른 언어도 함께 번역할 경우, 각 언어 탭을 순서대로 자동번역한 뒤 한 번에 [저장]하면 됩니다. 언어별로 따로 저장하지 않아도 됩니다.
07
게시 상태 변경
매거진 목록 → 상태 드롭다운
SCREENSHOT 07
매거진 목록에서 해당 언어 행의 상태 드롭다운으로 게시 상태를 변경합니다.
  • ✓목록에서 번역한 글 확인 (언어 컬럼으로 구분)
  • ✓상태 컬럼의 드롭다운 클릭
  • ✓상태 선택: 게시 / 비게시 / 검토필요
💡 게시 — FO 즉시 노출    비게시 — 저장만, FO 미노출    검토필요 — 담당자 검토 후 게시 예정
다중 언어 번역 시 반복 절차
5개 언어 전체 번역 시 STEP 3~5를 언어별로 반복합니다. 한 번의 복제 작성 화면에서 모든 언어 탭을 순서대로 번역한 뒤 저장하면 됩니다.
탭 순서 언어 코드 언어명 번역 방식 비고
1 EN 영어 직접 입력 / 원본 유지 원본 언어 — 자동번역 불필요
2 JA 일본어 자동번역 실행 STEP 3~5 반복
3 ZH 중국어 간체 자동번역 실행 STEP 3~5 반복
4 ZH-TW 중국어 번체 자동번역 실행 STEP 3~5 반복
5 ID 인도네시아어 자동번역 실행 STEP 3~5 반복
